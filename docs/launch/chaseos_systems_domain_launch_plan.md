# ChaseOS AI Domain Launch Plan

## Adopted domain strategy

`https://chaseos.ai` is the selected primary public launch domain. `chaseos.systems` is superseded as primary and may later be a secondary redirect, standards/ecosystem alias, or defensive domain if purchased. The exact `.com` is unavailable/not required for V1. Do not use get/try/use-prefixed domains, `chaseos.dev`, or `chaseos.app` as the public product domain.

## Launch phases

1. Domain lock: purchase/ownership/RDAP/DNS/brand sweep.
2. Static site + waitlist: homepage, waitlist, privacy/terms/security/open-core, Forge preview, Standards.
3. Repo/docs alignment: README/Foundation/Roadmap/V1 cutline/Feature Register/Forge docs.
4. Forge public static index: generate/upload/verify `/forge/index.json`; live fetch remains approval-gated.
5. Demo + distribution: `ChaseOS uses ChaseOS to launch chaseos.ai`.
6. Admin/beta operations: qualify users, invite first beta users, capture pack creator candidates.
7. Paid tests: setup sprint / Pro waitlist / Forge creator beta before full billing infrastructure.
