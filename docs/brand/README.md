# ChaseOS Brand Documentation

This folder holds the current ChaseOS brand foundation and preliminary visual identity references.

Use these files when updating README copy, website copy, product descriptions, UI labels, pitch material, design prompts, and future brand assets. These docs do not claim that the final logo, UI redesign, branded installer, or final design system is complete.

## Canonical Brand Rule

Use `ChaseOS` as the product name.

Avoid `ChaserOS`, `Chase OS`, `CHASE OS`, `Chase AI OS`, and `Chase Operating System` in current product copy unless the text is explicitly historical and labels the old usage as retired.
