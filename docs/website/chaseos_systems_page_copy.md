## Problem

AI work is scattered across chats, notes, repos, browser tabs, agents, and half-finished workflows. Every new session needs re-briefing. Every tool has its own memory. Every risky action needs manual checking.

ChaseOS gives the work one operating system.

## Product blocks

### ChaseOS Studio

The desktop command center for ChaseOS: inspect project memory, runtime status, workflow packs, approvals, docs, graph views, and provider/privacy state from one local-first interface.

### Chaser Forge

The marketplace and standards path for ChaseOS workflow packs, templates, extensions, agent presets, and operating kits. V1 starts with free/static preview paths and creator submission; paid marketplace, licensing, and automatic remote install remain future until proven.

### ChaseOS Standards

The pack, Forge index, approval, runtime, graph, source, outcome, entitlement, and managed-job contracts that make the ecosystem portable.

### Managed Agents

Future hosted runtime workers for users and teams who do not want to self-host. V1-facing copy should say `coming later` or `waitlist` only.
