# ChaseOS Launch Video Script and Shotlist

## Core narrative

ChaseOS uses ChaseOS to launch ChaseOS.

## 60–90 second teaser

1. Open on scattered AI work: chats, docs, repos, sources, browser workflows, approvals.
2. Cut to ChaseOS Studio: one local-first command surface.
3. Show knowledge graph: sources/projects/decisions connected.
4. Show agents/runtimes and Approval Center: power with boundaries.
5. Show missions/workflow packs and Chaser Forge preview.
6. End on `https://chaseos.ai` waitlist: ChaseOS Studio Early Access.

## 6–10 minute technical demo

- Demo fixture only, no private data.
- Studio homepage/graph/source/project view.
- Runtime/agent awareness read-only panel.
- Approval Center visibility.
- Mission/workflow pack example.
- Forge preview and `/forge/index.json`.
- Public website/waitlist.
- Honest blocked/future states.

## Optional developer walkthrough

Explain local-first repo/vault structure, standards examples, pack manifests, approval packet examples, no-secrets policy, and contribution/licensing posture.

## Must show

Studio, knowledge graph, sources/projects, agents/runtimes, missions/workflow packs, Approval Center, Forge preview, chaseos.ai waitlist, honest pending/blocked states.

## Must not show

Secrets, private paths, private API keys, personal info, fake payments, fake managed agents, unsupported browser automation, private repo internals that should not be public.
