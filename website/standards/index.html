Standards make AI work inspectable.

ChaseOS standards capture portable, public-safe shapes for workflow packs, Forge catalog entries, approval packets, graph objects, and outcome records.

Status: preview examples only. These files are draft scaffolds for V1 public review, not stable public APIs, live validators, managed entitlements, checkout, remote install, or hosted job authority.

Pack manifest

chaseos.pack.json describes a workflow pack, declared permissions, approval posture, and blocked external actions.

Forge index

chaseos.forge-index.json sketches a static pack catalog without live paid marketplace, managed entitlements, or remote-install jobs.

Approval packet

chaseos.approval.json shows the human-review gate before any external action can execute.

Graph object

chaseos.graph.json uses synthetic nodes and edges to show how projects, packs, approvals, and outcomes connect.

Outcome record

chaseos.outcome.json records local-only generated outputs and opt-in telemetry posture.

Additional drafts

agent, source, entitlement, and managed-job examples remain secondary preview scaffolds.

Preview boundary: no full stable API is claimed; no live managed entitlements, managed jobs, checkout, runtime credits, or untrusted third-party auto-install are enabled.
